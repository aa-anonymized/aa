aa: brute-force search for axioms for finite mathematical domains.

This version is for OSX; performance on other Unix-esque systems may be possible. Currently no Windows version.

Requires an installation of Prover9/Mace4:
https://www.cs.unm.edu/~mccune/mace4/download/

On first use, you will need to show the prover the location of your Prover9 installation: when prompted, open the version in the "bin" folder. 

Example input files are included. The executable file is "aa.jar".
